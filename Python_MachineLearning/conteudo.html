1. Regression (Regressão)

Método de aprendizado de máquina que possibilita identificar que variáveis tem impacto em um determinado tópico de interesse.

Você vai usar uma das técnicas de regressão para estimar o valor de venda de um apartamento, o número de chamadas em um Call-Center, ou o total de vendas em um determinado período. São os algoritmos mas simples do curso e permitem fazer uma estimativa adequada de um determinado valor em sem banco de dados, que seja relacionado as outras variáveis.

Existem diversos algoritmos de regressão, no Curso Python & Machine Learning estudaremos:

  • Regressão Linear.
  • Regressão Polinomial.
  • Regressão Múltipla.
  • Gradient Descent e alternativas.

2. Classification (Classificação)

Após a Regressão estudaremos a Classificação. Usamos a classificação para dizer se um determinado dado, pertence a uma classe específica.

São os algoritmos de classificação que usamos para definir se alguém pagará o empréstimo em dia, para nos livrar dos spams e que o Facebook usa para encontrar seus amigos em uma foto.

No Curso Python & Machine Learning estudaremos:

  • Regressão Logística.
  • Support Vector Machines (SVM).
  • Naíve Bayes.

3. Dimensionality Reduction (Redução de Dimensionalidade)

Um dos problemas que enfrentamos diariamente na aplicação de algoritmos de aprendizagem de máquina está no número de dimensões, variáveis, que os banco de dados possuem.

Muitos destes dados não causam impacto no objetivo desejado e precisam ser excluídos do processo por tornarem o processo de aprendizado muito complexo. Estas técnicas permitem selecionar as características dos nossos dados que facilitem o aprendizado e é isso que aprenderemos quando estudarmos Dimensionality Reduction no Curso Python & Machine Learning

No Curso Python & Machine Learning estudaremos:

  • Principal Component Analysis (PCA).
  • Kernel PCA.
  • Linear Discriminant Analysis (LDA).

4. Decision Tree (Árvores de Decisão)

Os algoritmos na classe de árvores de decisão são, em última análise, algoritmos de classificação. Estes são estes algoritmos que as empresas usam para prever as taxas de churn, o tempo de vida de um cliente e que os carros autônomos usam para classificar um pedestre ou um objeto.

No Curso Python & Machine Learning estudaremos:

  • Decision Trees.
  • Decision Tree Regression.
  • Random Forests.

5. k-Nearest Neighbors (kNN) (K-Vizinho mais próximo)

O Knn é um poderoso algoritmo de classificação que pode ser utilizado em dados que já estão classificados para permitir inferir onde um dado novo será classificado. Muito eficiente em situações em que é possível determinar uma distância entre os elementos já conhecidos, em problemas com poucos dados e pouca dimensionalidade. A análise de dados médicos é um exemplo típico do uso do KNN.

No Curso Python & Machine Learning veremos o KNN para detecção de cancer.

6. Ensemble Machine Learning (Aprendizado de Máquina em Conjunto)

São as técnicas mais quentes do mercado. Usaremos duas, ou três técnicas diferentes, em conjunto para melhorar o resultado dos algoritmos de aprendizagem.

No Curso Python & Machine Learning estudaremos:

  • Bagging.
  • Random Forests.
  • Extra Trees.
  • AdaBoost
  • Gradient Boosting Machine
  • XGBoost

Não, necessariamente nesta ordem

Aqui está o conteúdo do curso, mas não o planejamento das aulas. As aulas estão planejadas de forma a tirar o máximo proveito de cada dia. Pode ser que aconteça, por exemplo, que vejamos o básico do SVM em um dia e o detalhamento em outro. Isto foi planejado para que você possa aprender em uma sequência lógica e ir relacionando as técnicas e os algoritmos de forma integrada.
